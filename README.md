# HeadphonePorto 🎧

A sleek headphone e-commerce platform built with the Frontend.

## 🚀 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## ✨ Features
- User authentication
- Product listing and filtering
- Add to cart and checkout flow
- Admin panel
